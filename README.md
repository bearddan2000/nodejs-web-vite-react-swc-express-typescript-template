# nodejs-web-vite-react-swc-express-typescript-template

## Description
A react template without typecheck for the vite packager.

Feature
- swc skip typechecking

## Tech stack
- npm
- vite
- react
- express
- typescript

## Docker stack
- node

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
